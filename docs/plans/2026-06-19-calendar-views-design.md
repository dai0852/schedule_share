# 日・週・月カレンダー表示 設計

## 目的

営業チームの予定をGoogleカレンダーに近い日・週・月表示で閲覧できるようにし、予定の重なりと空き時間を直感的に把握できるようにする。

## 対象範囲

- 日表示: 選択した1日の時間軸グリッド
- 週表示: 選択日を含む月曜日から日曜日までの時間軸グリッド
- 月表示: 選択日を含む月の月間グリッド
- 「今日」「前へ」「次へ」による表示期間の移動
- 担当者フィルター、認証、予定更新、予定元の色分けの維持
- 選択中の表示期間に合わせたイベントAPIの再取得
- 終日予定の専用領域
- モバイルでの横スクロール

予定の作成、編集、削除、ドラッグ操作は対象外とする。イベントAPIと `NormalizedEvent` の契約は変更しない。

## 画面構成

### 共通ヘッダー

- 「今日」ボタン
- 前へ・次へボタン
- 表示対象期間の見出し
- 日・週・月の表示切替
- 取得件数

既存サイドバーには認証状態、担当者フィルター、予定更新、ログイン・ログアウトを残す。

### 日表示

- 左端に時刻、右側に1日分の列を表示する。
- 終日予定は時間軸の上に表示する。
- 時刻付き予定は開始・終了時刻に対応する縦位置と高さで表示する。
- 同じ時間帯に重なる予定は横方向に分割して表示する。

### 週表示

- 左端に時刻、右側に月曜から日曜までの7列を表示する。
- 各列のヘッダーに曜日と日付を表示する。
- 当日は背景色で識別する。
- 予定の縦位置、長さ、重複配置は日表示と同じ規則を使う。

### 月表示

- 月曜始まりの7列グリッドを表示する。
- 月の前後を含む週は補助色で表示する。
- 各日セル内に予定を開始時刻順で表示する。
- セル内に収まらない予定は件数表示にまとめる。

## 状態と期間計算

画面は次の状態を持つ。

- `mode`: `day | week | month`
- `selectedDate`: 表示基準日
- `selectedOwner`: 担当者フィルター
- `events`, `loading`, `error`: 取得状態

API取得期間は表示モードから算出する。

- 日: 選択日の開始以上、翌日の開始未満
- 週: 選択日を含む月曜日の開始以上、翌週月曜日の開始未満
- 月: 月間グリッドに表示する最初の月曜日以上、最後の日曜日の翌日未満

表示切替では `selectedDate` を維持する。前へ・次へは日表示で1日、週表示で1週、月表示で1か月移動する。「今日」は現在日へ戻す。

## コンポーネントとロジック

- `ScheduleApp`: 認証、フィルター、期間移動、API取得を管理する。
- `CalendarToolbar`: 期間移動、見出し、表示切替を担当する。
- `TimeGridCalendar`: 日・週の時間軸を描画する。
- `MonthCalendar`: 月間グリッドを描画する。
- カレンダー用純粋関数: 取得期間、表示日、終日判定、イベント配置、重複列を計算する。

既存の `NormalizedEvent` は変更せず、本文、参加者、会議URLを扱わない制約を維持する。

## イベント表示

- Google、Microsoft、Teamsは既存の色を使う。
- 表示内容は件名、担当者、場所、開始・終了時刻、予定元に限定する。
- 非公開予定はAPIで既に正規化された「予定あり」をそのまま表示する。
- 日付のみの開始・終了値を持つイベントを終日予定として扱う。
- 複数日にまたがる時刻付き予定は、表示日ごとに見える範囲を切り出して配置する。

## レスポンシブ対応

- デスクトップでは週表示の7列を画面幅内に配置する。
- 狭い画面では時間軸と週・月グリッドに最小幅を設け、カレンダー領域を横スクロール可能にする。
- サイドバーは既存どおりメイン領域の上に積む。

## エラーと読み込み

- 取得中は更新ボタンを無効化し、カレンダー上にも読み込み状態を示す。
- 取得失敗時は既存のエラー表示を維持する。
- 予定がない場合も空の時間軸または月間グリッドを表示し、「予定なし」を補助表示する。
- 直前の取得より古いレスポンスが後から返っても表示を上書きしないようにする。

## 検証

- 期間計算の日・週・月、月境界、年境界を単体テストする。
- 終日予定、複数日予定、重複予定の配置を単体テストする。
- 表示切替と期間移動が正しいAPI範囲を使うことをコンポーネントテストで確認する。
- `npm test`
- `npx tsc --noEmit`
- `npm run build`
- ブラウザで日・週・月、前後移動、今日、担当者フィルター、モバイル横スクロールを確認する。
